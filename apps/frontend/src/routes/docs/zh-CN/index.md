## 技术栈

| 层级 | 技术 |
|------|------|
| 运行时 | Bun 1.3.6+ |
| 前端框架 | SvelteKit + Svelte 5 |
| UI 组件 | shadcn-svelte |
| 后端框架 | Elysia |
| 数据库 | PostgreSQL + Drizzle ORM |
| 验证 | Zod v4 |
| 国际化 | Paraglide |
| 构建工具 | Turborepo |
